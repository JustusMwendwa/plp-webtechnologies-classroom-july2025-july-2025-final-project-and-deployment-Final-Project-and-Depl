JoyElite College Website – Project Documentation
1. Overview

The JoyElite College Website is a responsive multipage site built with HTML5, CSS3, and JavaScript.
It provides visitors with essential information about the college, available courses, and admission procedures, and includes a contact form for inquiries.

2. Objectives

Present JoyElite College in a professional, modern design.

Allow users to view courses and admission details easily.

Demonstrate core web technologies: semantic HTML5, advanced CSS3 styling, and JavaScript interactivity.

Deploy the website online so it’s accessible via a public URL.

3. Features

Five Pages:

Home: Hero section with slider and quick intro.

About: Details of JoyElite College’s mission and vision.

Courses: Grid of offered programs.

Admission: Instructions and downloadable forms.

Contact: Email/contact form with JavaScript validation.

Responsive Navigation Bar with links to all pages.

Image Slider showcasing campus life.

Decorative Footer with location, quick links, and social media icons (Font Awesome).

Smooth CSS Effects (gradients, hover states, transitions).
